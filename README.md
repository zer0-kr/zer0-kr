# Hey, I'm Zero 👋

**Security Compliance Engineer** who builds enterprise security frameworks from scratch — and makes them stick.

I work at the intersection of **Security**, **Cloud**, and **Compliance**, turning zero-security environments into certified, audit-ready organizations. Currently the sole security officer at a Series B startup, where I achieved ISMS certification within 15 months — with only 6 minor findings, no external consulting.

That work earned a **Commendation from the Minister of Science and ICT** 🏅

### What I Do

- 🔐 **Compliance & Certification** — ISMS / ISO 27001 / 27701 audit preparation, policy development (17 policies), regulatory response
- ☁️ **Cloud Security** — AWS/GCP vulnerability management (Prowler, Security Hub, SSM), IAM permission matrix, Cloud Custodian monitoring
- 🏗️ **Security Architecture** — Zero-trust endpoint (GWS + Keycloak), DAC (Teleport), endpoint security (Exosphere)
- ⚙️ **Automation** — Python-based workflow automation: onboarding, compliance enforcement, access control, cloud monitoring
- 🐛 **Offensive Security** — Bug bounty program operations, vulnerability assessment & remediation
